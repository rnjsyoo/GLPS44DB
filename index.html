GLPS 44 DB 관리보드 — 초기 설정 필요

아직 Supabase 연결 정보가 입력되지 않았습니다. 아래 순서대로 진행하세요.

  1. supabase.com 가입 → New Project 생성
  2. SQL Editor에서 supabase/schema.sql 실행 → 이어서 supabase/seed_students.sql 실행
  3. Authentication → Users → Add user로 나와 법인 계정(이메일/비밀번호) 2개 생성
  4. Project Settings → API에서 Project URL과 anon public key 복사
  5. 이 파일(index.html) 상단 CONFIG에 붙여넣고 저장 → 깃허브에 다시 업로드

자세한 내용은 함께 제공된 README.md 참고.
